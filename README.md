# Angular-7-CRUD-with-Firestore
Angular 7 CRUD with Firestore - Insert Update and Delete Using Firebase Project.

## Get the Code

```
$ git clone https://github.com/CodAffection/Angular-7-CRUD-with-Firestore.git
$ cd Angular-7-CRUD-with-Firestore/Angular 7
$ npm install
//run the app
$ ng serve
//Configure firebase project with your gmail account.
```
